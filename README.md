# FocalDesk

### Your desktop, finally organized.

FocalDesk is a Windows desktop overlay that turns icon chaos into a calm, organized workspace.

Your icons live inside elegant frosted glass frames and a sleek glass dock that are draggable, resizable, and themeable, while your wallpaper stays fully visible underneath.

It is not a launcher that replaces your desktop. It is your desktop, upgraded.

## 📦 Installation

1. Unzip the download anywhere you like, such as a **FocalDesk** folder in Documents or Program Files.
2. Run **FocalDesk.exe**.
3. No installer, no administrator rights, and nothing else to install. The .NET runtime is built in.
4. Paste your license key when prompted.
5. Enjoy a calmer desktop.

### Uninstall

1. Close FocalDesk from the tray icon (**Quit**).
2. Delete **FocalDesk.exe**.

Your desktop returns exactly to how Windows had it.
