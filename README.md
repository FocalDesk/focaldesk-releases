FocalDesk
Your desktop, finally organized.

FocalDesk is a Windows desktop overlay that turns icon chaos into a calm, organized workspace. Your icons live inside elegant frosted-glass frames and a sleek glass dock — draggable, resizable, themeable — while your wallpaper stays fully visible underneath. It's not a launcher that replaces your desktop. It is your desktop, upgraded.

Installing
Unzip anywhere you like (e.g. a FocalDesk folder in Documents or Program Files).
Run FocalDesk.exe. No installer, no admin rights, nothing else to install / the .NET runtime is built in.
Paste your license key when asked, and enjoy a calmer desktop.
To uninstall: close FocalDesk (tray icon → Quit) and delete the file. Your desktop returns exactly to how Windows had it.

“Windows protected your PC”?
The first launch may show a Windows SmartScreen warning — this is normal for new independent apps that haven't yet built a download reputation with Microsoft. Click More info → Run anyway. FocalDesk is safe: it has no telemetry, needs no account, and never connects to the internet on its own.

Updating
Updates within version 1 are free. Download the new zip from this page (or use About → Check for updates inside the app), close FocalDesk, and replace your FocalDesk.exe with the new one. Your license, frames, and themes are kept — they live in your Windows profile, not next to the exe.
